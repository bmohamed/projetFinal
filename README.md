# SIO 6014 : Applications web des SIO - Travail de session

**Auteurs**: Thomas Chabot NI : xxx-xxx-xxx, Mohamed Bouanane NI : xxx-xxx-xxx, Bechir Sellami NI : 111 052 979



**Description**: L'application "Simulateur" permet de simuler un système de production multiserveur comportant :
 - Une file d'attente 
 - Deux machines parallèles
 - Un noeud de décision qui permet de savoir si une pièce est complétée, à détruire ou à retravailller  

L'algorithme de simulation se base sur la méthode de simulation à évènement discret.


**Reference**: L'algorithme du moteur de simulation a été adapté en Dart à partir de l'algorithme développé en VB par M. Jean-Philippe Gagliardi dans le cadre du cours MQT-6013 Simulation de système. 

Le code servant à afficher le canvas a été adapté à partir de l'application balls développée par M. Dzenan Ridjanovic : https://github.com/dzenanr/balls
